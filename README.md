# TP-Link Homepage Clone
A responsive replica of the TP Link US homepage from scratch, using core HTML and CSS alongside vanilla Javascript to demonstrate fundamentals without the use of frameworks.

Several liberties have been taking to improve upon the current design of the existing live page.

## Frontend Techniques and Technoloogies
- **HTML5** - Semantic markup with proper heading hierarchy and accessibility considerations
- **CSS3** - Modern styling with advanced layout techniques for mobile responsiveness
 - Flexbox and Grid, media queries, animations
- **JavaScript (ES6+)** - Vanilla JavaScript for dynamic functionality
 - DOM manipulation, event handling, third-party library use
- **Swiper.js** - Third-party carousel library integration
 - Included documentation review to utilize particular parameters 

---

## Key Features
- **Image Carousel** - Auto-rotating image slider with fade transitions, pagination dots, and navigation controls
- **Horizontal Scrolling Reviews** - Updated carousel design with card structures and a subtle gradient overlay
- **Product Tabs** - Dynamic switching between product categories
- **Responsive Navigation** - Specific menu styling for smaller viewports
